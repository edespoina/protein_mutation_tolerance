# Protein Mutation Tolerance Analysis (6E60)

This project explores how many amino‑acid substitutions a small protein can tolerate before its fold is disrupted.

It includes:
- A systematic mutation strategy using BLOSUM62 + structural constraints
- AlphaFold2 predictions for mutant sequences
- RMSD‑based fold assessment
- A full scientific report (PDF)
- Python scripts used for mutation generation and structural analysis

This was completed as part of my MSc coursework to build foundational skills in computational structural biology.
